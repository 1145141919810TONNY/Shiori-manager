# Shiori-manager
Shiori manager（管理器）专用仓库

Shiori引擎本体：https://github.com/1145141919810TONNY/webgalgameengine

Shiori教学网页：https://1145141919810tonny.github.io/Shioriteaching/

# V1.0.2 更新内容（2026/6/26）

**新功能：游戏置顶 & 右键菜单**
- 右键游戏缩略图 → 弹出右键菜单，选择「置顶」可将游戏固定到列表最前面
- 再次右键 →「取消置顶」，游戏回到非置顶组最前
- 置顶不限数量，支持同时置顶多个游戏
- 置顶游戏右上角显示金色【置顶】标记
- 排序功能仅对非置顶游戏生效，置顶游戏始终保持原序并位于最前
- 拖拽排序时，置顶游戏只能与置顶游戏交换位置，非置顶游戏只能与非置顶交换
- 排序窗口同样支持右键菜单切换置顶/取消置顶，右侧同步显示【置顶】文字

**新功能：LEGUI / Locale Emulator 安装引导**
- 游戏详情栏新增「安装 Locale Emulator」引导按钮，当LE不可用时自动显示，一键跳转下载页面
- 底部状态栏新增「安装LEGUI」按钮，一键启动 LEInstaller.exe 安装器
- 启动安装器后自动弹出「LEGUI 安装提示」窗口（右侧定位，不遮挡安装界面），内置操作指引

**新功能：日语词典 & 排序方式**
- 接入 Kawazu（MeCab 形态素分析词典），日文游戏名称按五十音读音排序
- 内置 2500+ 常用汉字读音表（`KanjiReadingData`），词典外汉字自动回退
- 新增「按名称排序(A→Z)」按钮，混合语言智能排序（英文 → 假名 → CJK → 数字）
- 新增「排序规则说明」弹窗，详细解释分组优先级和排序策略

**视觉/体验优化**
- 排序窗口右键菜单整合（置顶 / 切换CJK语言），操作更直观
- 排序说明弹窗自动测量文本长度并适配窗口宽度

**Bug 修复**
- 修复启动游戏时窗口会卡死的问题（启动器不再等待进程退出，直接返回）
- 修复了语言包中部分遗漏的问题

# V1.0.0 （2026/6/22） 发布

## 功能特性

### 游戏管理
- **全盘扫描** — 自动搜索系统中所有 Shiori 引擎游戏并添加到管理器
- **本地 Games 扫描** — 一键扫描管理器同目录下 `games/` 文件夹
- **手动添加** — 手动选择游戏根目录，自动识别 Shiori 引擎
- **拖拽排序** — 长按游戏缩略图拖动，自由调整排列顺序，自动持久化保存
- **移除游戏** — 从管理器列表中移除入口（不删除游戏文件）
- **一键清除** — 批量清空管理器中的所有游戏入口

### 游戏启动
- **正式版启动** — 识别 `Shiori.exe` 并一键启动
- **调试版启动** — 识别 `Shiori_debug.exe`，并一键启动调试版
- **双版本检测** — 同时存在两个版本时可分别启动
- **端口冲突处理** — 自动检测 8080 端口占用，提示并提供关闭占用进程选项

### 版本更新
- **自动检查更新** — 启动时静默检查，发现新版本时工具栏显示红点提示
- **手动检查更新** — 点击「检查更新」按钮，查看版本差异和更新日志
- **一键跳转下载** — 弹窗确认后自动在浏览器打开 GitHub Releases 页面
- **版本异常检测** — 本地版本不在官方版本列表中时发出安全警告

### 多语言支持
- **简体中文** — 默认语言
- **繁體中文** — 繁体中文
- **English** — 英文
- **日本語** — 日文

### 其他功能
- **游戏详情面板** — 展示游戏名称、版本、作者、制作组、类型、简介、版权等元信息
- **桌面快捷方式** — 一键在桌面创建管理器快捷方式
- **版权协议弹窗** — 内置版权协议查看（多语言），第三方开源许可协议完整展示
- **玩家指南** — 内置玩家使用帮助文档
- **开发者注意事项** — 内置开发者技术文档
