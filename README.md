# V1.0.0 （2026/6/22） 发布

## 功能特性

### 游戏管理
- **全盘扫描** — 自动搜索系统中所有 Shiori 引擎游戏并添加到管理器
- **本地 Games 扫描** — 一键扫描管理器同目录下 `games/` 文件夹
- **手动添加** — 手动选择游戏根目录，自动识别 Shiori 引擎
- **拖拽排序** — 长按游戏缩略图拖动，自由调整排列顺序，自动持久化保存
- **移除游戏** — 从管理器列表中移除入口（不删除游戏文件）
- **一键清除** — 批量清空管理器中的所有游戏入口

### 游戏启动
- **正式版启动** — 识别 `Shiori.exe` 并一键启动
- **调试版启动** — 识别 `Shiori_debug.exe`，并一键启动调试版
- **双版本检测** — 同时存在两个版本时可分别启动
- **端口冲突处理** — 自动检测 8080 端口占用，提示并提供关闭占用进程选项

### 版本更新
- **自动检查更新** — 启动时静默检查，发现新版本时工具栏显示红点提示
- **手动检查更新** — 点击「检查更新」按钮，查看版本差异和更新日志
- **一键跳转下载** — 弹窗确认后自动在浏览器打开 GitHub Releases 页面
- **版本异常检测** — 本地版本不在官方版本列表中时发出安全警告

### 多语言支持
- **简体中文** — 默认语言
- **繁體中文** — 繁体中文
- **English** — 英文
- **日本語** — 日文

### 其他功能
- **游戏详情面板** — 展示游戏名称、版本、作者、制作组、类型、简介、版权等元信息
- **桌面快捷方式** — 一键在桌面创建管理器快捷方式
- **版权协议弹窗** — 内置版权协议查看（多语言），第三方开源许可协议完整展示
- **玩家指南** — 内置玩家使用帮助文档
- **开发者注意事项** — 内置开发者技术文档
